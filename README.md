# 🛠️ GitHub Copilot Advanced – Hands-on Workshop

**Microsoft Build 2026 | Hands-on Lab**

Welcome to the **GitHub Copilot Advanced** hands-on workshop! This lab is part of the [Microsoft Build 2026](https://msevents.microsoft.com/event?id=958083520) event and focuses on advanced techniques for leveraging GitHub Copilot in real-world development workflows.

---

## 📋 Workshop Overview

In this hands-on lab you will explore advanced GitHub Copilot capabilities including:

- **Getting Started with GitHub Copilot** – Completions, Chat, custom agents, Copilot CLI, custom instructions, prompt files, skills, and MCP servers
- **Spec-Driven Development (SDD)** – Make specifications (not vibes) drive what Copilot builds. End-to-end TypeScript/Node example
- **spec-kit** – An open-source toolkit by the GitHub team that formalizes the SDD loop for use with GitHub Copilot and other agentic AI coding tools
- **Multi-Agent Orchestration** – Orchestrating multi-agent development teams on top of GitHub Copilot

## ✅ Prerequisites

Before starting the workshop, please ensure you have the following:

| Requirement | Details |
|---|---|
| **GitHub Account** | With Copilot access – **scan the QR code provided on-site** to get GitHub Copilot Enterprise access for the duration of this workshop |
| **VS Code** | With GitHub Copilot + Copilot Chat extensions installed |
| **Node.js** | Version 20 or later |
| **GitHub CLI** | `gh` CLI with the `gh copilot` extension |

---

## 📖 What You'll Learn

By the end of this workshop, you will be able to:

1. Use GitHub Copilot's advanced features beyond basic code completions
2. Apply Spec-Driven Development to produce high-quality, predictable AI-generated code
3. Leverage spec-kit to formalize and scale your SDD workflows
4. Understand multi-agent orchestration patterns for complex development tasks
